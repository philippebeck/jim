# jim-js
JavaScript Interactive Microlibrary


# Installation

To install via npm :

$ npm install jim-js

To install via Bower :

$ bower install jim-js --save


# Overview

Jim is a microlibrary JavaScript who use the events for interactivity.

Jim is provided without any jQuery code line...
